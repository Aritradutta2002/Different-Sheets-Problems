## How To Use It In VS Code

- Open [SHEET.md](C:\Users\aritr\OneDrive\Desktop\JAVA_SHEET\JAVA_SHEET\SHEET.md) in one editor group.
- Open `src/Main.java` or the matching pattern folder in the second editor group.
- Each folder under `src` matches the sheet categories and pattern names.

## Naming Rule

- Category folders follow the sheet sections.
- Pattern folders follow the sheet pattern names, for example `converging_pointers`.
- Solved files stay inside the matching pattern folder.
- Package names match the folder path, for example `two_pointer_patterns.converging_pointers`.
